# Bhumi Assistant Context
- **Target OS:** Windows/macOS (Hybrid).
- **User Hardware:** 2017 MacBook Pro (OCLP) and a custom PC build.
- **Goal:** The assistant must be able to run heavy tasks (like Android ROM compilation) by triggering local shell scripts.
- **Privacy:** Always prefer Local LLM for private chats; only use API for web searches or heavy debugging.
- **Coding Style:** Clean, object-oriented Python. Use Docstrings for every function.
